gitで使うコマンド

// プッシュする時
$ git push origin [ブランチ名]

// プルする時
$ git pull origin [ブランチ名]

//ファイル全体のキャッシュ削除
$ git rm -r --cached .

//ファイルを指定してキャッシュ削除
$ git rm -r --cached [ファイル名]

// git 管理をやめる（該当のディレクトリにcdで移動してから下記を実行）
$ rm -rf .git/
